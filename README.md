Basics of GLFW and OpenGL in C++
================================

#### Steps to replicate:
  ```sh
  $ git clone https://github.com/gp-97/OpenGL-GLFW.git
  $ cd OpenGL-GLFW/lib
  $ rm -rf glfw
  $ git clone https://github.com/glfw/glfw
  $ cd ..
  $ mkdir build
  $ cd build
  $ cmake ..
  $ make
  $ ./OpenGL-GLFW
  ```
